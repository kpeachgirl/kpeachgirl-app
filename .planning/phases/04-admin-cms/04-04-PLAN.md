1. uploadImage(file: File, bucket: string, folder: string): Promise<string>
   - Generate unique path: `${folder}/${crypto.randomUUID()}-${file.name.replace(/[^a-z0-9.-]/gi, '_')}`
   - Use createClient() from lib/supabase/client.ts
   - Try direct upload first: supabase.storage.from(bucket).upload(path, file, { upsert: true })
   - If direct upload fails with auth error, fall back to signed URL upload:
     - supabase.storage.from(bucket).createSignedUploadUrl(path)
     - supabase.storage.from(bucket).uploadToSignedUrl(path, token, file)
   - Get public URL: supabase.storage.from(bucket).getPublicUrl(path)
   - Return publicUrl string
   - Reset file input value after upload (document pattern: caller handles this)
   - Per research pitfall #2: UUID paths prevent filename collisions

2. deleteStorageFile(bucket: string, path: string): Promise<void>
   - Extract path from full URL (remove the storage base URL prefix)
   - supabase.storage.from(bucket).remove([path])

3. triggerRevalidation(paths: string[]): Promise<void>
   - POST to '/api/revalidate' with { paths } body
   - Content-Type: application/json
   - No secret header needed for same-origin calls (per research pattern)
   - Fire-and-forget (don't block on response)

4. generateSlug(name: string): string
   - name.toLowerCase().replace(/\s+/g, '-').replace(/[^a-z0-9-]/g, '')

Export all functions. These will be imported by ModelEditor, GroupEditor, and ProfileFieldsTab in subsequent plans.
