**Submit button**: Full width, charcoal bg (disabled: sand), hover rose. Uppercase, 12px, font-weight-700. Disabled if required fields incomplete or submitting.

**Submission Handler**:
1. Set submitting = true
2. If file upload field has a File: upload to Supabase Storage 'submissions' bucket
   - File name: `${Date.now()}-${file.name}`
   - Use browser Supabase client: `supabase.storage.from('submissions').upload(fileName, file)`
   - Get public URL (NOTE: submissions bucket may be private -- if getPublicUrl doesn't work, store just the path and admin reads via signed URL later. For now, store the path as `submissions/${fileName}`)
   - If upload fails due to permissions: fall back gracefully, store null for id_photo_url, log warning
3. Insert into submissions table: `{ form_data: { ...formData, id_photo: undefined }, id_photo_url: uploadedPath, status: 'new' }`
4. On success: set done = true
5. On error: set error message, set submitting = false
